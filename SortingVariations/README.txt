Running in IntelliJ IDEA

1. Open IntelliJ IDEA and load the project.

2. Click Run > Edit Configurations.

	Set the following fields:

	Name: your preferred name

	Main class: Main 

	Working Directory: the project's location
 
	Program Arguments: 

	- <input folder> <output folder>: "Input/Lab 4 Input Files" "Output/Required Input_Output/"
	- <input folder> <output folder>: "Input/Generated_inputs" "Output/Generated_inputs_Output/"

	JDK: e.g. Java 22 SDK

3. Click Apply and OK.

4. Click Run
